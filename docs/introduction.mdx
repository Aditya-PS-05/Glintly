## What is Glintly?

Glintly is an AI-powered code generation and execution platform that allows developers to create, test, and deploy code through natural language interactions. Built with Next.js 15 and powered by advanced AI agents, Glintly provides secure sandboxed environments for code execution.

## How It Works

1. **Create a Project**: Start by creating a new project in the dashboard
2. **Describe Your Needs**: Use natural language to describe what you want to build
3. **AI Generation**: Our AI agents generate and execute code in secure sandboxes
4. **Review & Iterate**: Preview the results and iterate on your requirements
5. **Deploy**: Export your code or deploy directly from the platform

## Architecture Overview

Glintly is built on a modern stack:

- **Frontend**: Next.js 15 with React 19 and Tailwind CSS
- **Backend**: tRPC for type-safe APIs with PostgreSQL database
- **AI Layer**: Inngest agents powered by OpenAI GPT-4
- **Execution**: E2B sandboxes for secure code execution
- **UI**: Radix UI components for accessible design
